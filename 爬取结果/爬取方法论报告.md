## 一、可爬取的网站（3 家）

### 1. Amazon Jobs API ✅

**API 端点：** `https://amazon.jobs/en/search.json?city=Beijing&count=50&page=1`

**请求方式：** HTTP GET

**返回格式：** JSON，包含 `jobs` 数组，每个岗位有 `title`、`city`、`id`、`job_category`、`description`、`posted_date` 等字段

**爬取结果：** 7 个北京岗位，其中 2 个研发/技术类
- AI Agent Engineer, ARTS（Business Intelligence）
- Senior Software Dev Engineer, Kindle Device（Software Development）

**岗位链接格式：** `https://amazon.jobs/en/jobs/{job_id}/`

---

### 2. NVIDIA Workday API ✅

**API 端点：** `https://nvidia.wd5.myworkdayjobs.com/wday/cxs/nvidia/NVIDIAExternalCareerSite/jobs`

**请求方式：** HTTP POST，Body 为 JSON：`{"search":"","limit":20,"offset":0}`

**返回格式：** JSON，包含 `total`（总数）和 `jobPostings` 数组，每个岗位有 `title`、`locationsText`、`externalPath`、`postedOn` 等字段

**爬取方法：** 全量分页遍历 2000 个全球岗位，筛选 `locationsText` 包含 "China" 的记录

**爬取结果：** 86 个中国岗位，其中 10 个北京岗位（7 个研发/技术类）

**北京研发岗位示例：**
- Senior Autonomous Driving Software Engineer, L4 Planning
- Solution Architect, Jetson Platform Edge Computing
- Deep Learning Solution Architect
- Senior Software Engineer - Networking (RDMA and Doca)

**岗位链接格式：** `https://nvidia.wd5.myworkdayjobs.com{externalPath}`

---

### 3. Microsoft Careers（浏览器自动化）✅

**目标 URL：** `https://apply.careers.microsoft.com/careers?lc=Beijing&kw=software`

**爬取方式：** 使用 `catdesk browser-action` 浏览器自动化工具
1. 导航到搜索页面
2. 使用 `evaluate` 执行 JavaScript 提取页面中的岗位链接和文本
3. 解析标题、地点、发布时间

**爬取结果：** 页面显示 20 个全球岗位，其中 1 个北京岗位
- Digital Solution Engineer - D365 CE（中国，北京）

**限制：** Microsoft 搜索页面 `lc=Beijing` 参数未生效，返回的是全球岗位列表，需手动筛选

## 二、无法爬取的网站及原因

| 网站 | 原因 | 解决方案 |
|------|------|----------|
| BOSS直聘 (zhipin.com) | 反爬机制：返回 `{"code":37,"message":"您的环境存在异常"}` | 需要浏览器登录 + Cookie |
| 猎聘 (liepin.com) | 返回 HTML 而非 API JSON | 需要 Selenium/Puppeteer 渲染 |
| 中国移动 (job.10086.cn) | JavaScript SPA，数据动态加载 | 可用浏览器自动化（已验证可导航） |
| 中国电信 (job.chinatelecom.com.cn) | JavaScript SPA + 反爬脚本 | 需要浏览器自动化 |
| 工行 (job.icbc.com.cn) | JavaScript SPA | 需要浏览器自动化 |
| 中石油 (zhaopin.cnpc.com.cn) | 反爬混淆脚本 | 需要浏览器自动化 |
| Apple Jobs (jobs.apple.com) | API 返回 401 Unauthorized | 需要认证 Token |
| Google Careers | API 返回 404 | 无公开 API |
| Meta Careers | 无响应 | 无公开 API |
| Tesla Careers | Access Denied (Akamai WAF) | IP 被 WAF 拦截 |
| Oracle Careers | 重定向到 Oracle Cloud HCM | 需要 Oracle Cloud 认证 |
| Siemens Workday | 401 Unauthorized | 需要 Workday 认证 |
| SAP/Intel/AMD/Qualcomm Workday | 422 Unprocessable Entity | Workday 站点名称不正确 |

## 三、爬取方法总结

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| **HTTP API 直接调用** | 有公开 API 的网站（Amazon、NVIDIA Workday） | 快速、稳定、可批量 | 仅限有公开 API 的网站 |
| **浏览器自动化 (catdesk)** | JavaScript 渲染的 SPA 网站 | 可渲染动态内容 | 速度慢、需逐页处理 |
| **搜索引擎检索** | 无 API 的网站 | 覆盖面广 | 数据非实时、链接可能过期 |

**推荐策略：** API 优先 → 浏览器自动化 → 搜索引擎兜底
