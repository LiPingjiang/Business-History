# 企业史

> **一部记录历年各地企业的巨书。**
>
> 以企业为经，以地域为纬，以时间为轴，编织人类商业文明的全景图谱。

---

## 项目愿景

企业是现代文明的基本组织单元，一部企业史就是一部人类经济活动的微观通史。本项目的目标是系统性地记录、整理和研究全球各地企业的发展历程，从古代商帮到现代跨国公司，从手工作坊到科技巨头，构建一部兼具学术深度与可读性的企业通史。

## 项目范围

**地理维度：** 涵盖全球各大区域，以中国为核心，辐射东亚、东南亚、南亚、中东、欧洲、北美、拉美、非洲及大洋洲。

**时间维度：** 从古代商业萌芽（先秦至中世纪）到当代（截至2026年），贯穿古代、近代、现代、当代四个大时代。

**行业维度：** 包括但不限于金融、贸易、制造、矿业、交通、能源、通信、科技、零售、餐饮、医药、房地产、文化娱乐等。

**企业类型：** 从个体商户、家族企业、合伙组织，到股份公司、上市公司、跨国集团、平台企业，覆盖全谱系组织形态。

### 第一卷·总论

- [第一章：经济史方法论——以价格革命大辩论为例](./第一卷·总论/第一章_经济史方法论.md)
  — 以拉姆齐《十六世纪英格兰的价格革命》文选为核心案例，展示同一组历史数据在不同理论框架下如何产生截然不同的解释，探讨货币数量论与人口/需求论的方法论分歧及其对企业史研究的启示

### 第二卷·古代篇

- [第一章：都铎王朝与英国经济转型](./第二卷·古代篇/欧洲与地中海/第一章_都铎王朝与英国经济转型.md)
  — 基于 Peter H. Ramsey《都铎王朝的经济问题》(1963)，详述1485—1603年英国从传统农业到绵羊养殖与毛纺织业的行业演变，涵盖拉姆齐的核心学术立场（人口压力为价格革命首要变量）、货币大贬值、圈地运动、包买商制度及早期企业组织形态

### 第五卷·当代篇

- [智能时代与行业转型](./第五卷·当代篇/智能时代与行业转型.md)
  — 基于《贫穷的终结》（安妮·罗瑞），分析AI与自动化对美国和欧洲传统行业（制造业、零售业、运输业）的冲击，探讨技术性失业、零工经济、全民基本收入（UBI）与行业转型的出路

- [2026年北京外资企业名录（按国家/地区分类）](./第五卷·当代篇/2026年北京/北京外资企业名录_按国家分类.md)
  — 覆盖16个国家/地区约300家在京代表性外企，含入华时间、雇员规模、中国营收、空间分布与关键数据附录
- [2026年北京央企名录（按行业分类）](./第五卷·当代篇/2026年北京/北京央企名录_按行业分类.md)
  — 收录全部97家国资委监管央企，按11个行业分类，含总部、雇员、营收、招聘链接与世界500强排名

### 第六卷·行业专史

- [纺织业专史](./第六卷·行业专史/纺织业_专史.md)
  — 纵向梳理纺织业从都铎王朝羊毛业崛起到工业革命机械化生产，再到当代智能时代与自动化冲击的演变脉络，结合 Ramsey 与 Lowrey 两书，探讨新旧行业更替的历史规律

- [社会保障与福利制度专史](./第六卷·行业专史/社会保障与福利制度_专史.md)
  — 从都铎济贫法（1552年）到当代UBI全球实验（肯尼亚、芬兰、阿拉斯加等），梳理500年社会保障制度演变线，探讨制度对企业生态的深层影响
