# 企业史

> **一部记录历年各地企业的巨书。**
>
> 以企业为经，以地域为纬，以时间为轴，编织人类商业文明的全景图谱。

---

## 项目愿景

企业是现代文明的基本组织单元，一部企业史就是一部人类经济活动的微观通史。本项目的目标是系统性地记录、整理和研究全球各地企业的发展历程，从古代商帮到现代跨国公司，从手工作坊到科技巨头，构建一部兼具学术深度与可读性的企业通史。

## 项目范围

**地理维度：** 涵盖全球各大区域，以中国为核心，辐射东亚、东南亚、南亚、中东、欧洲、北美、拉美、非洲及大洋洲。

**时间维度：** 从古代商业萌芽（先秦至中世纪）到当代（截至2026年），贯穿古代、近代、现代、当代四个大时代。

**行业维度：** 包括但不限于金融、贸易、制造、矿业、交通、能源、通信、科技、零售、餐饮、医药、房地产、文化娱乐等。

**企业类型：** 从个体商户、家族企业、合伙组织，到股份公司、上市公司、跨国集团、平台企业，覆盖全谱系组织形态。
