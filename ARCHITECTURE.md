# 《企业史》编纂体例

> 本文件是《企业史》项目的编纂规范与写作指南，所有贡献者在撰写内容前须通读本文件。

---

## 一、编纂宗旨

以「实证、全景、可读」为三大原则：

**实证：** 每一条史实须有据可查，重要论断须标注来源。宁可存疑，不可杜撰。

**全景：** 兼顾宏观脉络与微观个案，不偏重某一地区或行业，力求呈现全球商业文明的整体图景。

**可读：** 学术研究与大众可读性兼顾。正文以流畅的现代汉语撰写，专业术语首次出现时加注说明。

## 三、企业条目体例

单个企业的研究条目（用于第七卷·企业列传）采用以下结构：

### 3.1 企业基本信息框

置于条目开头，以表格形式呈现：

| 字段 | 内容 |
|------|------|
| 企业名称 | 中文全称（外文原名） |
| 创立时间 | 年/月（如有据可查） |
| 创立地点 | 国家/地区/城市 |
| 创始人 | 姓名（如有多个，逐一列出） |
| 行业 | 主营行业分类 |
| 现状 | 存续/并购/破产/转型 |
| 总部 | 当前或最终总部地址 |
| 关键人物 | 历任核心领导人 |

### 3.2 正文结构

1. **概述** — 一段话概括企业全貌（创立背景、核心业务、历史地位）
2. **创立与早期** — 创立缘起、初始条件、早期发展
3. **发展历程** — 按时间阶段梳理关键节点
4. **商业模式** — 核心竞争力、盈利模式、战略选择
5. **重大事件** — 并购、危机、转型、上市等里程碑
6. **文化与管理** — 企业文化、组织架构、管理模式
7. **社会影响** — 对行业、地区、社会的影响
8. **评价与启示** — 学术评价与历史定位
9. **参考资料** — 引用来源列表

## 四、断代通史卷体例

第二至第五卷（断代通史）按以下层次组织：

1. **卷首总论** — 该时代全球商业环境概述
2. **分地区叙述** — 每个地区下设章节
3. **每章结构：**
   - 时代背景（政治、经济、技术环境）
   - 企业生态概览（主要企业类型与分布）
   - 代表性企业群像（选取典型企业展开）
   - 制度与政策（公司法、商业法规、产业政策）
   - 与其他地区的关联（贸易、投资、技术转移）
   - 本章小结

## 五、写作规范

### 5.1 语言

正文使用现代标准汉语（简体）。外国人名首次出现时附原文，如「洛克菲勒（John D. Rockefeller）」。外国企业名首次出现时附原文及常用简称，如「标准石油（Standard Oil）」。

### 5.2 时间表述

采用公历纪年为主。涉及中国传统纪年时括注公历，如「光绪二十一年（1895年）」。时间段统一用「至」连接，如「1840年至1945年」。

### 5.3 数据引用

财务数据、市场规模等量化信息须标注来源与年份。货币单位以原始记录为准，必要时括注折算值。如「1900年资产达2亿美元（约合今日60亿美元）」。

### 5.4 引注格式

采用脚注形式，格式参考：

- 书籍：作者，《书名》，出版社，年份，第X页。
- 论文：作者，「文章名」，《期刊名》第X期（年份），第X页。
- 网络资源：作者，「标题」，网站名，发布日期，URL。
- 档案：档案名称，卷宗号，馆藏机构。

### 5.5 客观性要求

对企业的评价须基于事实，避免简单褒贬。涉及争议事件时，呈现多方观点。不替企业做广告，不为商业利益背书。
