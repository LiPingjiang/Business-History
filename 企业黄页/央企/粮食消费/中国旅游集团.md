# 中国旅游集团（China Tourism Group）

> 《企业史》企业黄页

## 基本信息

| 项目 | 内容 |
|------|------|
| 企业名称 | 中国旅游集团 |
| 外文名称 | China Tourism Group |
| 企业类型 | 中央企业（央企） |
| 所属行业 | 旅游/酒店 |
| 成立时间 | 1928 |
| 总部 | 香港/海南 |
| 雇员规模 | 约4万人 |
| 营收 | 约800亿元 |

## 企业简介

中国最大的旅游央企，旗下含中旅/国旅/港中旅。
